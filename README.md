## 📋 プロジェクト概要

**GameGear Masters**は、PC周辺機器・ゲーミングデバイスに特化したアフィリエイトレビューサイトです。
FPS/TPSプレイヤーをターゲットに、マウス、キーボード、ヘッドセット、モニター、チェア、マウスパッドなど、
プロゲーマーも使用する高性能デバイスの詳細レビューとランキングを提供します。

### 🎯 サイトの目的

- **収益化**: Amazon・楽天アフィリエイトリンク経由での商品販売
- **SEO対策**: 「ゲーミングマウス おすすめ」「ラピッドトリガー キーボード」等の検索キーワードで上位表示を目指す
- **ユーザー価値**: 実機レビュー・スペック比較・選び方ガイドによる購買支援

## 🙋 よくある質問（FAQ）

### Q1: アフィリエイトリンクはどうやって取得するの？
**A**: 
1. [Amazonアソシエイト](https://affiliate.amazon.co.jp/)に登録
2. 商品ページでアフィリエイトリンクを生成
3. HTMLの`href="#"`部分を実際のリンクに置き換え

### Q2: 画像はどこから用意すればいい？
**A**:
- **公式サイト**: メーカー公式サイトから商品画像をダウンロード（利用規約確認）
- **Amazon商品画像**: Amazon Product Advertising API経由で取得
- **自分で撮影**: 実機を購入してオリジナル画像を撮影（最も信頼性高）

### Q3: SEO対策で最も重要なのは？
**A**: **コンテンツの質**が最重要。ユーザーが本当に知りたい情報（実機レビュー・比較・選び方）を提供することが、長期的なSEO成功の鍵です。

### Q4: 収益化までどれくらいかかる？
**A**: サイト公開後、SEO効果が出るまで**3-6ヶ月**が一般的。初期はSNS流入も活用しながら認知度を高めることが重要。
