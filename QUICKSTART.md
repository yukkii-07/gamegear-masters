# ⚡ クイックスタートガイド - 30分でサイト公開！

**初心者でもOK！** このガイドに従えば30分でサイトが公開できます。

---

## ステップ1: GitHubアカウント作成（5分）

### やること
1. https://github.com/ にアクセス
2. 右上の「Sign up」をクリック
3. メールアドレスを入力（フリーメールOK）
4. パスワードを設定
5. ユーザー名を決める（例: `gaming-reviewer-2024`）
6. メール認証を完了

**✅ 完了！** GitHubアカウントができました

---

## ステップ2: ファイルをアップロード（10分）

### やること

1. **リポジトリを作る**
   - https://github.com/new にアクセス
   - Repository name: `gamegear-masters` と入力
   - 「Public」を選択（誰でも見れる状態）
   - ✅「Add a README file」にチェック
   - 「Create repository」をクリック

2. **ファイルをアップロード**
   - 「uploading an existing file」をクリック
   - パソコンから以下のファイルをすべてドラッグ&ドロップ:
     - `index.html`
     - `gaming-mouse.html`
     - `gaming-keyboard.html`
     - `gaming-headset.html`
     - `gaming-monitor.html`
     - `gaming-chair.html`
     - `gaming-mousepad.html`
     - `js` フォルダごと
   - 「Commit changes」ボタンをクリック

**✅ 完了！** ファイルがアップロードされました

---

## ステップ3: サイトを公開（5分）

### やること

1. リポジトリのページで「Settings」タブをクリック
2. 左メニューから「Pages」をクリック
3. Source: `Deploy from a branch` を選択
4. Branch: `main` を選択
5. フォルダ: `/ (root)` を選択
6. 「Save」をクリック
7. **5-10分待つ**
8. ページを再読み込みすると、上部に緑色のバーで URL が表示される

**あなたのサイトURL:**
```
https://あなたのユーザー名.github.io/gamegear-masters/
```

**✅ 完了！** サイトが公開されました！🎉

---

## ステップ4: アフィリエイト登録（10分）

### Amazonアソシエイト

1. https://affiliate.amazon.co.jp/ にアクセス
2. 「無料アカウント作成」をクリック
3. Amazonアカウントでログイン（なければ作成）
4. 必要情報を入力:
   - **ウェブサイトのURL**: さっき公開したURL
   - **サイト名**: GameGear Masters
   - **サイトの説明**: ゲーミングデバイスのレビューサイト
5. 審査申請（通常1-3日で結果が来る）

**✅ 登録完了！** 審査結果を待ちましょう

### 楽天アフィリエイト

1. https://affiliate.rakuten.co.jp/ にアクセス
2. 楽天会員IDでログイン
3. サイト情報を登録
4. **即日利用可能！**

**✅ 完了！** 今すぐリンク作成できます

## 💡 よくある質問

### Q: HTMLがわからないんだけど...

**A**: 大丈夫！このサイトは編集しなくてもそのまま使えます。
慣れてきたら少しずつ変更していきましょう。

### Q: アフィリエイト審査に落ちたら？

**A**: 
1. サイトにもっとコンテンツを追加（記事を増やす）
2. 1ヶ月後に再申請
3. 楽天は審査なしで使えるので、そちらから始めてもOK

### Q: アクセスがゼロなんだけど...

**A**: 最初は当たり前！焦らずに：
1. Twitter/Xで宣伝
2. 友達にシェアしてもらう
3. 記事を増やす
4. 3ヶ月続ければGoogle経由でアクセスが増えてきます

### Q: 収益が出るまでどれくらい？

**A**: 
- 1ヶ月目: 0-1,000円（ほぼゼロ）
- 3ヶ月目: 1,000-5,000円
- 6ヶ月目: 5,000-30,000円
- 1年後: 30,000-100,000円以上も可能

**継続が大事！** 諦めずに続けましょう。
